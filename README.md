# eCommerce Web

Install TSLint extension for VS Code to lint source code.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# build
$ ng build # compiles application and outputs to dist folder

# start dev server with watch mode
$ ng serve

# production build
$ ng build --prod
```
## Test

```bash
# unit tests
$ ng test

# lint
ng lint

# e2e tests
$ ng e2e
```
